Blattlinie

Diese Website dient als Lernplattform für Blazor Server Technologie im Rahmen der Ausbildung an der HTBLA Leonding, Abteilung Informatik (78ABKIF). Sie bietet Kurse, Demos und Tutorials zu Blazor Server und modernen .NET-Entwicklungstechniken für Schüler:innen, Absolvent:innen und interessierte Entwickler:innen.

Grundlegende Richtung

Vermittlung von Fachwissen im Bereich der modernen Webentwicklung mit Blazor Server, Clean Architecture und Best Practices der Softwareentwicklung. Die Website richtet sich an Lernende und dient ausschließlich Bildungszwecken.
